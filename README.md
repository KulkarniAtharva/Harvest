# Harvest


Harvest , the best way to mark attendances of student in a much better organized way. With the most elegant user interface and latest Android OS support.
Harvest can be useful from academies towards much bigger schools, colleagues and universities to keep track of students.


Harvest app for mobile devices is rich in features that will make it easy to track attendance of your students.DeskTime can be used in an offline mode to later synchronize with the cloud.
